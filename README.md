# job-board-django
A full-stack Job Board platform built with Django that enables employers to post jobs and candidates to browse and apply. Implements CRUD operations, authentication-ready structure, and database integration with a scalable backend. Developed using Python, Django, SQLite, HTML, and CSS with a clean and responsive UI.
# Job Board Django Project

Features:
- Job posting
- Apply system
- Admin dashboard

Tech Stack:
- Django
- SQLite
- HTML, CSS
